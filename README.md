```bash
$ git clone git@github.com:MyBook/selfpub-ui.git
$ cd selfpub-ui
$ yarn install
$ yarn storybook
```

Open your browser and visit http://localhost:9001, see more at [Development](https://github.com/MyBook/selfpub-ui/wiki/Development).
